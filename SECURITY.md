# 보안 · 책임 있는 사용 (SECURITY)

이 확장은 **교육/테스트 목적**의 시각적 데이터 추출 도구입니다. 아래는 권한·데이터 처리·안전장치·사용자 책임을 정리한 문서입니다.

## 1. 데이터 처리 — 모든 것이 로컬, 외부 전송 없음

- 추출/설정 상태는 **`chrome.storage.local`** 에만 저장됩니다: `uc_job`(수집 결과·진행), `uc_consent`(오리진별 ToS/robots 확인), `uc_selection`/`uc_field_pick`(선택 중간상태).
- **외부 서버로 보내는 데이터가 없습니다.** 클라우드 동기화·분석·텔레메트리 없음.
- 네트워크 요청은 단 하나 — 대상 도메인의 **`/robots.txt`** fetch(확인용)뿐입니다.
- 결과는 사용자가 누른 **다운로드(CSV/JSON)** 로만 로컬 파일에 저장됩니다(data URL → `chrome.downloads`). 저장 위치는 다운로드 폴더(또는 그 하위 지정 폴더).
- 로그인/쿠키/비밀번호에 접근하지 않습니다.

## 2. 권한(permissions) 사용 근거

| 권한 | 용도 |
|---|---|
| `activeTab`, `scripting` | 현재 탭에 추출/선택 로직 주입(정적 + 폴백 동적) |
| `storage`, `unlimitedStorage` | 위 로컬 상태 저장(수집 누적 여유) |
| `downloads` | CSV/JSON 로컬 저장 |
| `tabs` | 잡 오케스트레이션(`tabs.onUpdated`로 페이지 로드 감지, 메시지 라우팅) |
| `sidePanel` | 선택 중에도 닫히지 않는 컨트롤 패널 |
| `host_permissions: <all_urls>` | 사용자가 연 탭에서만 동작. 확장 페이지의 robots.txt cross-origin fetch에도 필요 |

`<all_urls>`는 광범위하지만 **실제 동작은 사용자가 선택한 탭에 한정**됩니다. MVP 단순성을 위한 선택이며, 향후 연습 도메인으로 좁히거나 `activeTab` 동적 주입으로 전환할 수 있습니다(README 한계 참고).

## 3. 책임 있는 추출 — 내장 안전장치(우회 불가)

- **robots.txt 준수**: 추출 전 `lib/robots.js`(RFC 9309)로 현재 경로를 판정. 🔴 Disallow면 사용자가 명시적으로 "이해했고 계속한다"를 눌러야만 진행, 🟠(확인 불가)는 보수적 차단.
- **지연 하한 2000ms / 페이지 상한 20**: popup 입력 보정 + **background에서 무조건 재클램프**(프로필 import로도 우회 불가).
- **ToS 게이트**: 첫 추출 전 ToS 확인 체크 필요. consent는 **오리진 스코프**로 저장되어 사이트 간 우회 차단. 프로필 export에는 확인 상태를 직렬화하지 않음(import 후 항상 재확인).
- 위 강제 경로는 `unit.js`/`run.js`로 검증됩니다.

## 4. 명시적 비목표 (이 도구가 하지 않는 것)

봇 차단 우회, 프록시, User-Agent 위조, TLS 핑거프린트 조작, CAPTCHA 처리, 로그인 자동화 — **일절 구현하지 않습니다.**

## 5. 사용자 책임

- 대상 사이트의 **robots.txt와 이용약관(ToS)** 을 준수하는 범위에서만 사용하세요.
- 학습은 **연습 전용 공개 사이트**로: `books.toscrape.com`, `quotes.toscrape.com`, `scrapethissite.com`.
- 실제 상업 사이트를 무단 대량 수집하지 마세요. robots가 🔴면 그 사이트는 자동 수집을 **금지**한 것입니다(도구가 막는 게 정상).
